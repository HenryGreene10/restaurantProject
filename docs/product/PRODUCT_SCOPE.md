# Product Scope (Phase 1 emphasis)

- Validate multi‑tenant ordering foundation with real restaurants on PWA.
- Admin dashboard: menu management, order list, hours/pauses.
- Kitchen display: reliable, glanceable, resilient to connectivity issues.
- Payments: Stripe Connect Standard onboarding; SaaS Billing configured.
- Notifications: SMS fallback for all critical events; push optional.
- AI assistant reserved for Phase 2 once real data exists.
